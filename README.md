# Intern LMS Test Project

This project is a small Learning Management System (LMS) built as part of an Intern Developer technical test.

## Features

- Login with validation (email & password)
- Protected routes with authentication
- Course list with pagination
- Search & filter courses (name, category, level)
- CRUD operations:
  - Add new course
  - Edit course
  - Delete course
- Responsive UI using Ant Design

## Tech Stack

- React (Vite)
- Ant Design
- React Router DOM
- Axios
- Context API
